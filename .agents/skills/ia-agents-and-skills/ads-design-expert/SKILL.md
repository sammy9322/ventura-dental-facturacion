# Ads Design Expert Skill

## Proposito

Especializado en crear publicidad digital efectiva para plataformas
de redes sociales y medios digitales.

## Capacidades

### Tipos de Anuncios

1. **Awareness**: Anuncios de conocimiento de marca
2. **Consideration**: Anuncios de consideracion (carousel, video)
3. **Conversion**: Anuncios de conversion (ofertas, CTA)
4. **Retargeting**: Anuncios personalizados para usuarios previos

### Proceso Creativo

1. **Brief**: Analizar objetivos, presupuesto, audiencia
2. **Concepto**: Generar ideas que conecten emocionalmente
3. **Diseno**: Ejecutar con jerarquia visual clara
4. **Copy**: Escribir textos persuasivos con CTAs claros
5. **Optimizacion**: Probar variaciones para mejorar rendimiento

### Elementos Clave

- **Headline**: Titulo llamativo (max 5-7 palabras)
- **Visual**: Imagen o video que capture atencion
- **Copy**: Mensaje claro y conciso
- **CTA**: Llamada a la accion obvia

## Buenas Practicas

- Usar contraste de color para destacar
- Incluir texto sparingly (no mas de 20% de la imagen)
- Mantener simplicidad - un mensaje por anuncio
- Testear multiples variaciones
- Usar colores de marca consistentemente

## Codigos de Color Recomendados

- Facebook: Usar azul (#1877F2) como color de marca
- Instagram: Paleta visual de la marca
- Google Ads: Blanco con acentos de color para contraste
